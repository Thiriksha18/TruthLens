# TruthLens: Trust Scoring System for LLM Responses

## Overview
TruthLens is an AI-based system that evaluates the trustworthiness of responses generated by Large Language Models (LLMs).

## Features
- Response trust scoring
- Confidence estimation
- Fact-checking support
- Explainable AI outputs

## Technologies Used
- Python
- Machine Learning
- Natural Language Processing
